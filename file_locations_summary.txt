KEY DETECTION LOGIC
================================================================================

V2 Format Detection (renderer.py:244-250):
  if section has "headline" AND "tts_text" fields → V2 FORMAT
    → Use Playwright + HTML templates
    → Use FFmpeg Ken Burns + xfade composition
  else → V1 FORMAT (legacy)
    → Use PIL Pillow
    → Use MoviePy simple fade

Slide Duration Calculation (composer.py:255-296):
  if sections have "tts_text" field:
    → Proportional: len(tts_text) / total_chars * duration
  else:
    → Equal: duration / num_slides

Short Video Detection (composer.py:180-182):
  if _sections.json exists in frames_dir → V2 FORMAT
    → Try Remotion v3
    → Fallback to FFmpeg v2
  else → V1 FORMAT
    → Use MoviePy legacy
